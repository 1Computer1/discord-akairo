### About
A bot framework for Discord.js v11.  
- Command processing.
- Argument parsing.
- Command inhibitors.
- Event listeners.
- Reloadable everything.
- SQLite support.

### Installation
discord-akairo: `npm install discord-akairo --save`  
discord.js: `npm install discord.js --save`  
sqlite: `npm install sqlite --save`
